# U.S. Meal-Delivery Company Directory — by State (with CEO / Leadership)

A directory of U.S.-based meal-delivery, meal-kit, and prepared-meal companies, organized
by the state of their headquarters, with the current CEO / top executive where it could be
verified. Built to support outreach/lead-gen (e.g., comparable to **Factor**, **Eat Clean
To Go**, and **Magic Kitchen**, which the brief called out).

- **Compiled:** 2026-06-20
- **Method:** Verified via live web research (company sites, press releases, trade press,
  LinkedIn, Crunchbase). Prospecting databases (Apollo, Vibe Prospecting) were not usable
  for this run (Apollo free-plan gating / Vibe out of credits), so this is web-sourced.
- **Confidence:** `High` = corroborated by 2+ sources; `Med` = single decent source;
  `Low / Unverified` = company confirmed but leadership name not publicly confirmed (never
  guessed — left blank/"Unverified").
- **Note on executives:** Leadership changes often. For acquired brands, the listed exec may
  be a brand GM/founder or roll up to the parent company's CEO (noted inline). Treat exec
  names as "last verified," and re-confirm before any formal outreach.

**Directory size:** **~493 companies** — the national/segment-leading brands **plus** a deep
roster of smaller local operators in **all 50 states + DC**. The **CSV
(`meal_delivery_directory.csv`) is the complete, canonical list** and is sorted by state; the
per-state tables in this Markdown cover the first wave of entries (the CSV contains the rest).

**Verified emails:** `meal_delivery_directory.csv` includes **Email** and **Email Status**
columns. ~350 named executives across the full 493-company list were run through Apollo
enrichment; **111 returned a verified business email** (status `verified`/`extrapolated`).
The remaining named owners weren't in Apollo's database (typical for very small local
operators) — for those, use the public website/contact email. A handful of matched emails
are the exec's other venture (status still shows verification for the person) — sanity-check
the domain before company-specific outreach. ~250 Apollo credits used in total.

### Field key
- **HQ City / State** — headquarters location (not necessarily where they ship).
- **CEO / Top Exec** — most senior verifiable leader; *Unverified* = company is real but no
  public exec name (never guessed).
- **Confidence** — High (2+ sources) / Med (single source) / Low (company only).

*Re-verify any executive name and contact details immediately before outreach — meal-delivery
leadership and ownership change frequently (acquisitions, restructurings, founder exits).*
